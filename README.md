# Sahan Yasiruth Portfolio

Bold dark-mode portfolio for an AI/ML engineering undergraduate focused on NLP, local LLM applications, RAG systems, and multimodal AI.

## Run Locally

```bash
npm install
npm run dev
```

If PowerShell blocks `npm.ps1`, use:

```bash
cmd /c npm run dev
```

## Build

```bash
npm run build
```

## Update Content

- Profile, links, and image: `src/data/profile.js`
- Projects and metrics: `src/data/projects.js`
- Skills: `src/data/skills.js`

Fill in `email`, `linkedin`, and `resume` in `src/data/profile.js` when those final URLs are ready.
